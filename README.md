## Descrição
Uma aplicação desenvolvida em Java que consome uma API de CEP. Logo, essa aplicação permite o usuário consultar todas as informações de uma CEP pesquisado por ele. O projeto também possui funcionalidas de ordenar os CEPs pesquisados, ver as estatítiscas das pesquisas e consultar de forma inteira ou resumida o CEP.

# :hammer: Funcionalidades do projeto
- `Funcionalidade 1`: Pesquisar CEP;
- `Funcionalidade 2`: Pesquisar CEP Completo;
- `Funcionalidade 3`: Listar CEPs pesquisados;
- `Funcionalidade 4`: Listar CEPs ordenados;
- `Funcionalidade 5`: Exibir Estatísticas;
